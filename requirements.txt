# --- Online Deployment Quick Start ---
# 1. Database Setup:
#    - Host a PostgreSQL database (e.g. on Supabase, Neon, RDS, or Render PostgreSQL).
#    - Import and run the SQL schema located in `./database/schema.sql`.
#
# 2. Backend Environment Variables (.env inside ./backend):
#    PORT=3001
#    DATABASE_URL=postgresql://username:password@hostname:5432/dbname
#    JWT_SECRET=your_jwt_secret_token
#    JWT_EXPIRES_IN=8h
#    SMTP_HOST=smtp.gmail.com
#    SMTP_PORT=587
#    SMTP_USER=your.email@gmail.com
#    SMTP_PASS=your_gmail_app_password
#    IMAP_HOST=imap.gmail.com
#    IMAP_PORT=993
#    CLIENT_URL=https://your-frontend-deployment-url.com
#
# 3. Frontend Environment Variables (.env.local inside ./frontend):
#    NEXT_PUBLIC_API_URL=https://your-backend-deployment-url.com/api
#
# 4. Installation & Build:
#    - Backend: cd backend && npm install && npm start
#    - Frontend: cd frontend && npm install && npm run build && npm start
